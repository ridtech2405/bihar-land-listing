# 🎲 Mama Bhanja Lottery Management System

A Django-based Lottery Management System developed to manage lottery operations, user authentication, and related features through a simple and user-friendly web interface.

---

## 📌 Features

- User Authentication
- Lottery Management
- Dashboard
- Responsive Interface
- Django Admin Panel
- Static & Media File Management

---

## 🛠️ Technologies Used

- Python 3.x
- Django
- HTML
- CSS
- JavaScript
- Bootstrap
- SQLite

## ⚙️ Installation

Clone Repository

```bash
git clone <repository-url>
```

Move into project

```bash
cd mama_bhanja
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py migrate
```

Create Admin

```bash
python manage.py createsuperuser
```

Run Project

```bash
python manage.py runserver
```

Visit

```
http://127.0.0.1:8000/
```
